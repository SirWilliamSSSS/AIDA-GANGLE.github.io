รายละเอียดการยินยอม

เพื่อให้เป็นไปตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) ทางบริษัท/ผู้ให้บริการ ขอแจ้งให้ท่านทราบว่า ข้อมูลส่วนบุคคลของท่านที่เราได้รับ จะถูกเก็บรวบรวม ใช้ และเปิดเผย ภายใต้หลักเกณฑ์ดังนี้

1. ข้อมูลส่วนบุคคลที่เราเก็บรวบรวม
เราจะเก็บข้อมูลต่อไปนี้เมื่อท่านใช้บริการผ่าน LINE OA หรือระบบวิเคราะห์ภาพของเรา:
- ชื่อ-นามสกุล
- เพศ
- อายุ
- รูปภาพที่ท่านส่งเข้ามา (เช่น ภาพผิวหน้า หรือร่างกาย)
- ข้อมูลที่เกี่ยวกับผลการวิเคราะห์ - ข้อมูลผู้ใช้งาน LINE เช่น LINE User ID, ชื่อใน LINE, รูปโปรไฟล์ (ถ้ามี)
- วันเวลา และประวัติการใช้งานระบบ
2. วัตถุประสงค์ในการใช้ข้อมูล
- วิเคราะห์ภาพที่ท่านส่งเข้ามาโดยใช้ระบบ AI
- แนะนำสินค้า บริการ หรือแนวทางดูแลสุขภาพ/ผิวพรรณ (ถ้ามี)
- ปรับปรุงประสบการณ์การใช้งานให้เหมาะสมยิ่งขึ้น
- ติดต่อกลับในกรณีจำเป็น
3. ระยะเวลาในการเก็บรักษาข้อมูล
- ข้อมูลส่วนบุคคลจะถูกเก็บไว้ ไม่เกินระยะเวลาที่จำเป็น
- เพื่อวัตถุประสงค์ที่ระบุไว้ หรือจนกว่าท่านจะร้องขอให้ลบข้อมูล
4. การเปิดเผยข้อมูล
- เราจะไม่เปิดเผยข้อมูลของท่านให้บุคคลภายนอก เว้นแต่ :
ได้รับความยินยอมจากท่าน
เป็นไปตามข้อกฎหมาย หรือคำสั่งของหน่วยงานราชการ 5. สิทธิของเจ้าของข้อมูล
- ท่านมีสิทธิ : ขอเข้าถึง / แก้ไข / ถอนความยินยอม
ในการเก็บ ใช้ หรือเปิดเผยข้อมูล ขอให้ลบ หรือทำลายข้อมูลส่วนบุคคล
โดยสามารถติดต่อเราได้ตามช่องทางที่ระบุไว้
6. มาตรการรักษาความปลอดภัย
- เราดำเนินการรักษาความมั่นคงปลอดภัยของข้อมูลของท่าน ด้วยมาตรการทางเทคนิคและองค์กร เช่น การเข้ารหัส (encryption), การจำกัดการเข้าถึง, และการควบคุมผู้ใช้งานระบบอย่างเหมาะสม
7. ช่องทางติดต่อ
- หากท่านมีคำถามเกี่ยวกับนโยบายนี้ หรือต้องการใช้สิทธิของเจ้าของข้อมูล กรุณาติดต่อ :
ฝ่ายดูแลข้อมูลส่วนบุคคล
